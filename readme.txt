== Description ==

### WooKit Plugin

Wookit is a plugin for WooCommerce. It adds a lots of settings on the wordpress customizer where you can adjust the design and parameters of
WooCommerce main areas like shop page, single product page, checkout page, etc...Almost every WooCommece html element could be stylized,
changing colors, sizes, font-size, font family, spaces and margins and you can see it the live changes under customizer without reload the page.
Also, with Wookit PRO you can add new features to your shop, like a woocommerce product carousel, inserting it on page positions where shortcodes
or widgets can't reach, such as above the main woocommerce single products, or below it, even inside the content.


### How to use

After you succesfully installed, on your wp customizer menu you will find a new option named "WooKit".
It's very intuitive, so just adjust values as you want and you will see changes instantly.


### WooCommerce Highlighted Features (WooKit 1.3)


#### WooCommerce Carousel for Single Product Page
Display On sale, Top Rated, Best Selled or related products in a carousel inside your products page. A highly customizable feature.


== Installation ==

Automatic installation is the easiest option as WordPress handles the file transfers itself and you do not need to leave your web browser. To do an automatic install of WooKit, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.
